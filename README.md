# Password Generator and Strength Checker

A Java command-line tool for generating secure passwords and passphrases, and for evaluating the strength of any password. Built as an independent project to apply object-oriented design principles alongside practical cybersecurity concepts like cryptographically secure randomness, character-set control, and weak-credential detection.

## Features

- **Secure password generation** using `SecureRandom` (not `Math.random()`), with full control over:
  - Password length
  - Character types included (uppercase, lowercase, digits, symbols)
  - Exclusion of visually ambiguous characters (e.g. `l`, `I`, `1`, `O`, `0`, `o`, `8`, `B`)
  - Exclusion of adjacent repeating characters
- **Passphrase generation** (e.g. `Cyberpunk-Beach-Monitor7`) with control over:
  - Number of words
  - Custom separator character
  - Capitalization of each word
  - An appended random digit
- **Password strength checking** for any password, whether generated by this tool or typed in manually, based on:
  - Length
  - Character variety (uppercase, lowercase, digits, symbols)
  - A known-weak-password check against a small reference list
- **Simple CLI menu** for generating a password, generating a passphrase, checking password strength, or exiting.

## How It Works

The program is split into seven collaborating classes, each with a single responsibility:

| Class | Responsibility |
|---|---|
| `MainPasswordTool` | CLI driver â€” displays the menu, collects user input, and calls into the other classes |
| `PasswordCriteria` | Data holder for password generation settings (length, character types, exclusions) |
| `PasswordGenerator` | Builds a character pool from `PasswordCriteria` and generates a password using `SecureRandom` |
| `PassphraseCriteria` | Data holder for passphrase generation settings (word count, separator, capitalization, number) |
| `PassphraseGenerator` | Builds a passphrase from `WordBank`'s word list using `SecureRandom` |
| `StrengthChecker` | Evaluates any password's strength (length, variety, known-weak-password check) |
| `WordBank` | Static reference data â€” a word list for passphrases and a list of common weak passwords |

### Design notes

- Each "criteria" class (`PasswordCriteria`, `PassphraseCriteria`) is a simple data holder â€” validation logic (e.g. requiring at least one character type to be selected) lives in `MainPasswordTool`, keeping the data classes focused purely on storing settings.
- `StrengthChecker` and `PasswordGenerator` intentionally keep their own separate copies of small reference data (like the symbol set), rather than depending on each other. This keeps `StrengthChecker` fully self-contained so it can evaluate *any* password, not just ones this tool generated.
- Ambiguous-character filtering and adjacent-repeat filtering are handled with backward iteration and re-roll loops respectively, to avoid index-shifting bugs and infinite-loop risk.

## Running It

Compile and run with the JDK:

```
javac *.java
java MainPasswordTool
```
